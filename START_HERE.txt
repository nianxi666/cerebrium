🆘 遇到问题？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【问题】401 Unauthorized
→ 检查 API Key 是否正确设置

【问题】Timeout
→ 减少推理步数或帧数

【问题】400 Bad Request
→ 检查参数格式，num_frames 必须是 4n+1 格式

【问题】视频质量差
→ 增加推理步数和指导尺度

更多帮助 → 查看 QUICK_START.md 的故障排除部分
